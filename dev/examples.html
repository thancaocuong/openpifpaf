Example predictions from the COCO val set. These are the first images with more than five person annotations that were shared with CC-BY-2.0. These examples are not cherry-picked.

All examples are predicted with the fast model shufflenetv2k16w.
For more accurate results, try --checkpoint=shufflenetv2k30w.

The examples below are generated with the shown command which is documented in Prediction.
